       1 # cFabricPC Project Status & Architecture
       2
       3 This file tracks the current state of the `cFabricPC` repository, its core architecture, and ongoing development tasks.
       4
       5 ## Current Version: 0.3.0 (2026-04-17)
       6
       7 ### Core Architecture
       8 cFabricPC is a JAX-based predictive coding library organized around three main pillars:
       9 1. **Nodes**: Independent computational units (latents, projections, activations).
      10 2. **Wires (Edges)**: Connections between nodes defining the graph topology.
      11 3. **Updates**: Local Hebbian learning and iterative inference algorithms.
      12
      13 ### Key Features
